# SEO Checklist

- [ ] Title tag includes primary topic term naturally (e.g. "pharmaceutical
      supply chain traceability") without keyword stuffing
- [ ] Meta description (150-160 chars), factual, no sales language
- [ ] URL slug is short and descriptive (e.g. /ra-001-pharma-traceability)
- [ ] H1 matches article title; H2s match major section headers exactly
      (Executive Summary, Why This Problem Matters, etc.)
- [ ] Images (diagrams) have descriptive alt text, not "diagram1.png"
- [ ] Internal links to related articles once 2+ exist
- [ ] External links to primary sources (GS1, FDA, WHO, etc.) — these build
      topical trust signals
- [ ] Page load: images optimized/compressed before publishing
- [ ] Structured data: Article schema (see ai-discoverability-checklist.md
      for the overlap with this)
