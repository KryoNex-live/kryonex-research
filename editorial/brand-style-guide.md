# KryoNex Research — Brand Style Guide (v1.0, fixed)

This file is the single source of truth for visual identity. Do not introduce
variations elsewhere — if something isn't covered here, add it here first,
then use it.

## Logo
Wordmark only: "KryoNex Research" — no icon, no mark, no symbol.
- Set in Plex Sans (or Inter), single weight (Medium), no gradients, no effects.
- Minimum clear space: height of the "K" on all sides.
- Never stretch, rotate, or recolor outside the two approved brand colors.

## Colors
| Role | Name | Hex |
|---|---|---|
| Primary | Graphite | #1B2430 |
| Accent | Steel Blue | #4A7A96 |
| Background | Off-white | #FAFAF8 |
| Text (body) | Graphite | #1B2430 |
| Muted / captions | Slate Gray | #6B7684 |

Only these five. No secondary accent color, ever — this is the discipline
that keeps every asset (diagrams, covers, site) looking like one publication.

## Typography
| Use | Typeface | Fallback |
|---|---|---|
| Headings | Plex Sans | Inter, system-ui |
| Body text | Plex Serif | Source Serif Pro, Georgia |
| Code / labels / diagram text | Plex Mono | Consolas, monospace |

## Diagram Style
- Line diagrams only: thin borders (1-1.5px), no drop shadows, no 3D, no skeuomorphism.
- Palette: Graphite for boxes/text, Steel Blue for flows/highlights, Slate Gray for secondary elements.
- Swimlanes for multi-actor systems (e.g., Manufacturer / Distributor / Pharmacy / Regulator).
- Every diagram is numbered ("Figure N") with a one-line caption and a legend if more than 2 symbol types are used.
- Font in diagrams: Plex Mono, one size for labels, one size for captions.
- Source files (editable) live in each article's diagrams/ folder; exported PNG/SVG accompanies them.

## Footer (use verbatim on every article/page)
© {year} KryoNex Research · kryonex.live · Reference architectures are
provided for informational purposes; implementation details will vary
by organization.

## Author Format (use verbatim structure)
Prepared by {Full Name} — KryoNex Research
{One-line role, e.g. "Industrial Automation Engineer"}

No bio paragraph, no social links, no calls to action.

## Language Rules
- No superlatives ("best," "leading," "revolutionary," "world-class").
- No first-person plural marketing voice ("we help you...").
- No product mentions or CTAs inside article body. A single neutral
  "About KryoNex Research" note may appear only at the very end of the piece,
  after References.
